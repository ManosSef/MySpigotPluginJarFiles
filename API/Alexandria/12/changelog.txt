BUILD 12

Additions
- me.manossef.api.alexandria.commands.ReloadConfigCommand: contains the implementation for a command that takes in "reload" as the first argument and reloads its plugin's config.yml
	- ReloadConfigCommand#executor(Plugin): returns a CommandExecutor for the above command implementation, using the given plugin's config.yml
	- ReloadConfigCommand#tabCompleter(): returns a TabCompleter that tab completes "reload" for the first argument
- me.manossef.api.alexandria.config.ConfigurationRecord: an interface intended to be implemented by records used for storing plugins' settings
	- Contains the methods #logOptions(Logger) and #saveToFile(FileConfiguration)
- All methods in me.manossef.api.alexandria.config.OptionValidations have been given variants that start with "recheck" instead of "check" and don't disable the plugin when an option is invalid (but still send the message), which are all meant to be used on a config reload instead of the initial load.
