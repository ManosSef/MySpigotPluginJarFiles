Changes
- SpigotMCUpdateChecker now extends SimpleUpdateChecker, since they were already very similar
	- SpigotMCUpdateChecker no longer overrides #checkForUpdates() and #getPluginName()
- YamlFile#saveDefaultConfig() is no longer run when a YamlFile object is constructed, so that plugins can choose whether to save the default file when creating a new one
- TabCompletions methods that take objects as parameters now immediately throw NullPointerException once if called with null arguments instead of throwing errors when tab completing arguments
